# اپلیکیشن شیوه‌نامه‌های جشنواره

این پروژه یک برنامه آفلاین اندروید است که ۱۲ شیوه‌نامه PDF، صفحه نخست پوستر جشنواره، نمایشگر داخلی PDF و جست‌وجوی یکپارچه فارسی را در خود دارد.

## ساخت APK با GitHub

1. در GitHub یک مخزن جدید بسازید.
2. تمام محتویات این پوشه را در ریشه مخزن بارگذاری کنید.
3. وارد زبانه **Actions** شوید و اجرای Workflow را تأیید کنید.
4. فرایند **Build Android APK** را باز و گزینه **Run workflow** را بزنید.
5. پس از پایان اجرا، در بخش **Artifacts** فایل `festival-guidelines-apk` را دریافت کنید.
6. فایل ZIP را باز کنید؛ فایل `app-debug.apk` آماده نصب آزمایشی است.

برای انتشار عمومی در فروشگاه‌ها باید نسخه release امضاشده ساخته شود؛ نسخه debug برای نصب مستقیم و آزمایش مناسب است.
